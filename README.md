# misc-c

Miscellaneous C language scripts.

## execd (WIP)

Ad-hoc supervisior daemon.
Based on [D. J. Bernstein](https://cr.yp.to/djb.html)'s [daemontools](https://cr.yp.to/daemontools.html).

### Usage

``` sh
execd /run/execd/yes /usr/bin/yes Hello World!
```
